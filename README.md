# Crystal of Atlan DMG Calculator
 
 A simple web-based damage calculator for **Crystal of Atlan**.
 
 ## Features
 
 - **Damage calculation** with a fast, in-game inventory inspired grid-style UI
 - **Defense / mitigation inputs**
 - **Local persistence** (keeps your inputs after refresh)
 - **Responsive layout** for desktop and mobile
 
 ## Tech Stack
 
 - **React + TypeScript**
 - **Vite**
 
 ## Getting Started
 
 ### Install
 
 ```bash
 npm install
 ```
 
 ### Run locally
 
 ```bash
 npm run dev
 ```
 
 Then open the URL shown in the terminal (usually `http://localhost:5173`).
 
 ## Scripts
 
 ```bash
 npm run dev
 npm run build
 npm run preview
 ```
